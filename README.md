SafePK is a cyber awareness and digital safety platform designed to help users understand online threats, improve their security habits, and learn basic cybersecurity concepts in simple language.

---

## 🚀 About SafePK

SafePK provides beginner-friendly cybersecurity resources, safety tools, and awareness guides to help people protect themselves from common online risks.

The goal is simple:

> Make cybersecurity easy and understandable for everyone.

---

## ✨ Features

### 🔐 Security Tools

- Password Generator
- Password Strength Checker
- Security Checklist
- Phone Safety Check

Users can check their security habits and get guidance to improve their online protection.
